# Nexa Digital Solutions - Corporate Website Specification

## 1. Project Overview
- **Project Name**: Nexa Digital Solutions Website
- **Type**: Multi-page Corporate Website (5 pages)
- **Core Functionality**: Showcase IT services, portfolio, company info, and enable client contact
- **Target Users**: B2B clients seeking software development & IT services

## 2. Pages Structure

### Pages List
1. **index.html** - Home Page (main landing page)
2. **about.html** - About Us Page (company info, team, values)
3. **services.html** - Services Page (detailed services)
4. **projects.html** - Projects/Portfolio Page (all projects)
5. **contact.html** - Contact Page (contact form, info)

### Shared Components
- Header with logo and navigation
- Footer with quick links, contact info, social icons

## 3. UI/UX Specification

### Layout Structure
- **Container**: Max-width 1200px, centered
- **Responsive Breakpoints**:
  - Desktop: 1200px+
  - Tablet: 768px - 1199px
  - Mobile: < 768px

### Visual Design
- **Colors**:
  - Primary: #0A1628 (Dark Blue)
  - Secondary: #1a2d4a
  - Accent: #0EA5E9 (Sky Blue)
  - Accent Light: #38BDF8
  - Background: #FAFBFC (White majority)
  - Card Background: #FFFFFF
  - Text Primary: #0F172A
  - Text Secondary: #64748B
- **Typography**:
  - Headings: "Outfit" (Google Fonts)
  - Body: "DM Sans" (Google Fonts)
